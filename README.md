# MVP-PUC-RIO-1
Trabalho mercado financeiro 1

RESUMO
Este trabalho apresenta uma análise quantitativa completa do comportamento intraday do mercado WINFUT utilizando Databricks e arquitetura Bronze–Silver–Gold. Foram realizadas etapas de ingestão, limpeza, preparo dos dados e análise estatística, incluindo volatilidade por horário, probabilidade de retorno, clusterização de regimes (baixa, média e alta volatilidade) e identificação de padrões operacionais. Os resultados oferecem evidências claras sobre horários favoráveis para tendência, reversão, scalp e períodos de consolidação.
