# Objetivo do MVP

## Contexto
Este MVP tem como objetivo analisar dados intraday do contrato futuro de índice (WINFUT),
utilizando um pipeline de dados em nuvem para identificar padrões de volatilidade,
retorno e regimes de mercado ao longo do dia.

## Problema a ser resolvido
Operadores e analistas enfrentam dificuldade em identificar, de forma objetiva,
quais períodos do dia apresentam maior risco (volatilidade) e melhor assimetria
(retorno esperado), bem como regimes distintos de comportamento do mercado.

## Perguntas de Negócio
1. Quais horários do dia apresentam maior volatilidade média?
2. Existe diferença significativa de retorno médio entre faixas de horário?
3. É possível identificar clusters de candles com comportamentos semelhantes?
4. Esses clusters apresentam padrões distintos de risco e retorno?
5. Como esses padrões podem apoiar decisões operacionais e estratégicas?

## Valor do MVP
O MVP entrega uma base analítica que permite:
- Apoiar decisões de trade intraday
- Identificar regimes de mercado
- Servir como base para dashboards analíticos e modelos futuros
